Menurut saya validate status dalam setiap automation test adalah hal yang penting. Dengan adanya status code kita jadi tahu bagaimana keadaan api web kita. apakah responnya successfull, redirect, terjadi error di client, terjadi error di server dll. Dengan mengetahui status code api web kita jadi lebih mudah untuk menentukan langkah kedepaannya. Semisal ada error, kita pun jadi lebih mudah untuk mengidentifikasi di mana terjadinya error.
