prettier memiliki kegunaan untuk membuat code yang kita punya menjadi rapih, konsisten, dan mudah atau enak dibaca secara otomatis.
menurut saya file ini penting karena salah satu keuntungan yang bisa didapatkan dengan menerapkan eksistensi ini adalah membuat code kita tersusun rapih secara otomatis ketika kita mengklik tombol ctrl+save. jadi kita tidak perlu bersusah payah mengatur tampilah code kita dengan cara manual. selain itu, file ini juga membuat kita memiliki tampilan kode yang konsisten sesuai dengan format yang kita atur. kegunaan lainnya adalah ketika kita bekerja secara team, semua format code yang dituliskan oleh masing-masing anggota menjadi selaras yang membuat setiap anggota team bisa lebih enak melihatnya.
